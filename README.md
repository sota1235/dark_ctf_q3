Dark CTF Admin
====

Admin tool to manage the flags.

### Description

If you are admin user, you can look the flag.

Login with below informations.

```
user: admin
pass: {enter admin password}
```

### Requirement

- PHP7

### Usage

```
php -S localhost:8080
```

### Install

```
mv .env.sample .env
php composer.phar install
```
